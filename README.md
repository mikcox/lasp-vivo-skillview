# Yeoman + PHP
A sample workaround to use [yeoman](http://yeoman.io) with php. 

## Prerequisites
    $ sudo apt-get install php5-cgi

## Try it
    $ git clone git@github.com:revathskumar/yeoman-php.git
    $ cd yeoman-php
    $ npm install
    $ bower install
    $ grunt server
